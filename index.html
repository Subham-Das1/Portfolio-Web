Projects

Memory Game

The Memory Game is an interactive emoji-matching game. Players flip two face-down cards at a time, aiming to find matching pairs. If the two selected cards show the same emoji, they get hidden, indicating a successful match. The goal is to match all pairs before running out of moves to win.

Simple Interest Calculator

The Simple Interest Calculator, built using HTML, CSS, and JavaScript, allows users to easily compute the interest on a principal amount by inputting the rate and time period. This tool offers a quick and efficient way to calculate simple interest in real-time.

To-Do List

The To-Do List application is a tool built with HTML, CSS, and JavaScript. It allows users to add, complete, and delete tasks while ensuring date validation. Features include responsive design, and smooth animations, making task management simple and visually appealing.

Mood Calendar

A mood calendar is a tool for tracking and reflecting on emotions over time. Users mark their mood each day using emojis, colors, or notes. It helps identify patterns and trends in emotional well-being, offering insights for self-awareness, mental health management, or even team dynamics.

Number Guessing Game

A Number Guessing Game is an interactive activity where players try to guess a randomly generated number within a specified range. With each guess, players receive hints to guide them closer, making it a fun and engaging way to test intuition and logic.

Maze Game

A Maze Game is an interactive challenge where players navigate through different pathways to reach the goal within less time. Each maze offers a unique puzzle that tests problem-solving skills. It provides a fun and rewarding experience for players of all ages.
